### 풀이 방법
- 능률에 따른 소요일을 계산하기 위해 다음과 같은 식을 사용함
$$day = \frac{(progress-100)}{speed}$$
- 이를 통해 days 리스트에 각 작업의 소요일을 저장함.
- 저장된 days의 좌측부터 탐색을 시작해 내림차순으로 정렬된 숫자 배열의 길이를 저장함.
- 만약, 좌측 숫자보다 큰 숫자일 경우 숫자 배열의 탐색을 재시작하여 숫자 배열의 길이를 다시 탐색하여 저장함.
- 위 과정을 반복하여 문제를 해결함.
